## ⚡ CRITICAL RULES BEFORE YOU START ANY SPRINT

### Reanimated 4 + Skia 2.6 — The Two Laws

**Law 1: No Reanimated shared values accessed directly inside Skia canvas.**  
Everything that drives Skia must go through `useDerivedValue`.

```typescript
// ❌ WRONG — crashes in Reanimated 4 + Skia 2.6
const progress = useSharedValue(0);
<Circle cx={50} cy={50} r={progress.value * 50} />

// ✅ CORRECT
const progress = useSharedValue(0);
const radius = useDerivedValue(() => progress.value * 50);
<Circle cx={50} cy={50} r={radius} />
```

**Law 2: `useSharedValue` and `useDerivedValue` must be called at the top level of a component — never inside loops or conditions.**  
If you have N particles, each particle must be its own component that receives its seed as a prop. The parent creates N components; each component owns its own shared values.

```typescript
// ❌ WRONG — violates Rules of Hooks
const particles = Array(50).fill(0).map((_, i) => {
  const x = useSharedValue(Math.random() * width); // CRASHES
  return x;
});

// ✅ CORRECT
// Parent:
const seeds = useMemo(() => Array(50).fill(0).map(() => ({
  x: Math.random() * width,
  y: Math.random() * height,
  speed: 0.5 + Math.random() * 1.5,
})), []);
return seeds.map((seed, i) => <SingleParticle key={i} seed={seed} />);

// Child component (each has its own hooks at top level):
function SingleParticle({ seed }: { seed: ParticleSeed }) {
  const x = useSharedValue(seed.x);
  const y = useSharedValue(seed.y);
  // ...
}
```

### Step 3: The stable ParticleCanvas pattern

Replace your `components/particles/ParticleCanvas.tsx` with this pattern that handles theme switching without crashing:

```typescript
// components/particles/ParticleCanvas.tsx
import React, { useState, useCallback, useEffect } from 'react';
import { View, LayoutChangeEvent } from 'react-native';
import { useAppStore } from '../../store/appStore';
import { useTheme } from '../../hooks/useTheme';
import DustParticles from './DustParticles';
import SnowParticles from './SnowParticles';
import RainParticles from './RainParticles';
import OceanWave from './OceanWave';

interface ParticleCanvasProps {
  breathPhase?: 'inhale' | 'holdIn' | 'exhale' | 'holdOut' | 'idle';
  opacity?: number; // 0–1, defaults to 1
}

export default function ParticleCanvas({
  breathPhase = 'idle',
  opacity = 1,
}: ParticleCanvasProps) {
  const [dimensions, setDimensions] = useState({ width: 0, height: 0 });
  const [ready, setReady] = useState(false);
  const activeTheme = useAppStore((s) => s.activeTheme);
  const theme = useTheme();

  // KEY: remount particle component on theme change to prevent
  // stale shared values from old theme accumulating
  const [mountKey, setMountKey] = useState(0);
  useEffect(() => {
    setMountKey((k) => k + 1);
  }, [activeTheme]);

  const onLayout = useCallback((e: LayoutChangeEvent) => {
    const { width, height } = e.nativeEvent.layout;
    if (width > 0 && height > 0) {
      setDimensions({ width, height });
      setReady(true);
    }
  }, []);

  // Map theme to particle type from themes.ts config
  const particleTypeMap: Record<string, string> = {
    rajasthan: 'dust',
    himalaya: 'snow',
    kerala: 'rain',
    assam: 'rain',
    andaman: 'ocean',
  };
  const particleType = particleTypeMap[activeTheme] ?? 'dust';

  const renderParticles = () => {
    if (!ready) return null;
    const { width, height } = dimensions;
    const props = { width, height, theme, breathPhase };

    switch (particleType) {
      case 'dust':  return <DustParticles key={`dust-${mountKey}`} {...props} />;
      case 'snow':  return <SnowParticles key={`snow-${mountKey}`} {...props} />;
      case 'rain':  return <RainParticles key={`rain-${mountKey}`} {...props} />;
      case 'ocean': return <OceanWave key={`ocean-${mountKey}`} {...props} />;
      default:      return <DustParticles key={`dust-${mountKey}`} {...props} />;
    }
  };

  return (
    <View
      onLayout={onLayout}
      pointerEvents="none"
      style={{
        position: 'absolute',
        top: 0, left: 0, right: 0, bottom: 0,
        opacity,
      }}
    >
      {renderParticles()}
    </View>
  );
}
```

### Step 4: The stable single-particle pattern (DustParticles as reference)

This is the **correct Skia 2.6.2 + Reanimated 4 pattern** for a particle component. Use this as your template for all 5 particle types:

```typescript
// components/particles/DustParticles.tsx
import React, { memo, useMemo } from 'react';
import { Canvas, Circle } from '@shopify/react-native-skia';
import { useSharedValue, useDerivedValue, withRepeat,
         withSequence, withTiming, useEffect as useReanimatedEffect
} from 'react-native-reanimated';

interface ParticleProps {
  width: number;
  height: number;
  theme: any;
  breathPhase?: string;
}

interface DustSeed {
  startX: number;
  startY: number;
  radius: number;
  driftAmpX: number;
  driftAmpY: number;
  driftSpeedX: number;
  driftSpeedY: number;
  baseOpacity: number;
}

// Single particle — owns its own hooks safely at top level
const DustParticle = memo(function DustParticle({
  seed, color, breathPhase,
}: { seed: DustSeed; color: string; breathPhase?: string }) {
  const t = useSharedValue(seed.startX * 100); // offset start phase

  // Start continuous animation on mount
  useReanimatedEffect(() => {
    t.value = withRepeat(
      withSequence(
        withTiming(1000, { duration: (seed.driftSpeedX * 1000) / 1 }),
      ),
      -1, // infinite
      false
    );
  }, []);

  const cx = useDerivedValue(() => {
    const phase = (t.value % 1000) / 1000;
    // Gentle drift: sine wave + breath phase influence
    let breathOffset = 0;
    if (breathPhase === 'inhale') breathOffset = -8 * Math.sin(phase * Math.PI);
    if (breathPhase === 'exhale') breathOffset = 8 * Math.sin(phase * Math.PI);
    return seed.startX + seed.driftAmpX * Math.sin(phase * Math.PI * 2) + breathOffset;
  });

  const cy = useDerivedValue(() => {
    const phase = (t.value % 1000) / 1000;
    return seed.startY + seed.driftAmpY * Math.cos(phase * Math.PI * 2 * seed.driftSpeedY);
  });

  const opacity = useDerivedValue(() => {
    const phase = (t.value % 1000) / 1000;
    return seed.baseOpacity * (0.5 + 0.5 * Math.sin(phase * Math.PI * 2));
  });

  return (
    <Circle cx={cx} cy={cy} r={seed.radius} color={color} opacity={opacity} />
  );
});

export default function DustParticles({ width, height, theme, breathPhase }: ParticleProps) {
  const PARTICLE_COUNT = 35;

  const seeds: DustSeed[] = useMemo(() =>
    Array(PARTICLE_COUNT).fill(0).map(() => ({
      startX: Math.random() * width,
      startY: Math.random() * height,
      radius: 1 + Math.random() * 2.5,
      driftAmpX: 20 + Math.random() * 40,
      driftAmpY: 15 + Math.random() * 30,
      driftSpeedX: 4000 + Math.random() * 8000,
      driftSpeedY: 0.3 + Math.random() * 0.7,
      baseOpacity: 0.15 + Math.random() * 0.35,
    })),
  [width, height]);

  if (width === 0) return null;

  return (
    <Canvas style={{ width, height }}>
      {seeds.map((seed, i) => (
        <DustParticle
          key={i}
          seed={seed}
          color={theme.particle}
          breathPhase={breathPhase}
        />
      ))}
    </Canvas>
  );
}
```

**Apply the same structure to SnowParticles, RainParticles, and OceanWave** — each uses its own `SingleParticle` memo component that owns its shared values at the top level.

#### Step 2: Upgrade `components/breathing/BreathingOrb.tsx`

This is the full replacement. It adds:
- Phase-reactive gradient (inner color blends between `orbInhale`/`orbExhale`/`orbHold`)
- Glow ring that leads on inhale, trails on exhale
- All driven by `phaseProgress` from your existing `useBreathing` hook

```typescript
// components/breathing/BreathingOrb.tsx
import React from 'react';
import { View } from 'react-native';
import { Canvas, Circle, RadialGradient, vec } from '@shopify/react-native-skia';
import { useDerivedValue, interpolate, Extrapolation } from 'react-native-reanimated';
import type { SharedValue } from 'react-native-reanimated';

type BreathingPhase = 'inhale' | 'holdIn' | 'exhale' | 'holdOut' | 'idle';

interface BreathingOrbProps {
  phase: BreathingPhase;
  // phaseProgress is a Reanimated SharedValue (0→1 within current phase)
  phaseProgress: SharedValue<number>;
  theme: {
    orb: string;
    orbGlow: string;
    orbInhale: string;
    orbExhale: string;
    orbHold: string;
  };
  size?: number; // diameter, default 220
}

export default function BreathingOrb({
  phase, phaseProgress, theme, size = 220,
}: BreathingOrbProps) {
  const center = size / 2;
  const maxRadius = size * 0.42;  // orb max size
  const minRadius = size * 0.22;  // orb min size
  const glowExtra = size * 0.12;  // how much bigger the glow ring is vs orb

  // --- Orb radius (driven by phase + phaseProgress) ---
  const orbRadius = useDerivedValue(() => {
    const p = phaseProgress.value;
    switch (phase) {
      case 'inhale':
        return interpolate(p, [0, 1], [minRadius, maxRadius], Extrapolation.CLAMP);
      case 'holdIn':
        return maxRadius;
      case 'exhale':
        return interpolate(p, [0, 1], [maxRadius, minRadius], Extrapolation.CLAMP);
      case 'holdOut':
      case 'idle':
        return minRadius;
      default:
        return minRadius + (maxRadius - minRadius) * 0.5;
    }
  });

  // --- Glow ring radius (leads on inhale, lags on exhale) ---
  const glowRadius = useDerivedValue(() => {
    const p = phaseProgress.value;
    const base = orbRadius.value + glowExtra;
    switch (phase) {
      case 'inhale':
        // glow is ~15% ahead of orb
        const aheadInhale = interpolate(p, [0, 0.15, 1], [0, glowExtra * 0.6, glowExtra], Extrapolation.CLAMP);
        return minRadius + aheadInhale + (maxRadius - minRadius) * p;
      case 'exhale':
        // glow lags ~15% behind orb (contracts after orb)
        const lagExhale = interpolate(p, [0, 0.15, 1], [glowExtra, glowExtra * 0.6, 0], Extrapolation.CLAMP);
        return maxRadius - (maxRadius - minRadius) * Math.max(0, p - 0.15) + lagExhale;
      default:
        return base;
    }
  });

  // --- Inner color (blends between phase colors) ---
  const centerColor = useDerivedValue(() => {
    // Return the static color string for the current phase
    // Skia RadialGradient accepts string colors
    switch (phase) {
      case 'inhale':   return theme.orbInhale;
      case 'holdIn':   return theme.orbHold;
      case 'exhale':   return theme.orbExhale;
      case 'holdOut':  return theme.orbExhale;
      default:         return theme.orb;
    }
  });

  // --- Glow ring opacity (pulses during holdIn) ---
  const glowOpacity = useDerivedValue(() => {
    if (phase === 'holdIn') {
      // Gentle shimmer: 0.25 → 0.45 pulse using phaseProgress as sine driver
      return 0.25 + 0.2 * Math.sin(phaseProgress.value * Math.PI * 3);
    }
    if (phase === 'holdOut') return 0.12;
    if (phase === 'inhale')  return 0.15 + phaseProgress.value * 0.25;
    if (phase === 'exhale')  return 0.4 - phaseProgress.value * 0.2;
    return 0.15;
  });

  return (
    <View style={{ width: size, height: size }}>
      <Canvas style={{ width: size, height: size }}>
        {/* Glow ring (behind the orb) */}
        <Circle
          cx={center}
          cy={center}
          r={glowRadius}
          color={theme.orbGlow}
          opacity={glowOpacity}
        />

        {/* Main orb with radial gradient */}
        <Circle cx={center} cy={center} r={orbRadius}>
          <RadialGradient
            c={vec(center, center)}
            r={orbRadius}
            colors={[centerColor, theme.orb]}
            positions={[0, 1]}
          />
        </Circle>
      </Canvas>
    </View>
  );
}
```

```typescript
// components/timer/CircularTimer.tsx
import React from 'react';
import { View } from 'react-native';
import { Canvas, Circle, Path, Group, ClipRect,
         Rect, Skia, useDerivedValue } from '@shopify/react-native-skia';
import { useSharedValue, withRepeat, withTiming,
         useEffect as useReanimatedEffect, interpolateColor } from 'react-native-reanimated';
import type { SharedValue } from 'react-native-reanimated';

interface CircularTimerProps {
  progress: SharedValue<number>; // 0 (start) → 1 (complete)
  size?: number;
  theme: {
    primary: string;
    primaryLight: string;
    background: string;
    cardBorder: string;
    text: string;
  };
}

export default function CircularTimer({
  progress, size = 240, theme,
}: CircularTimerProps) {
  const cx = size / 2;
  const cy = size / 2;
  const r = size * 0.44;

  // Wave phase — scrolls continuously for live-water feeling
  const wavePhase = useSharedValue(0);
  useReanimatedEffect(() => {
    wavePhase.value = withRepeat(
      withTiming(Math.PI * 2, { duration: 3000 }),
      -1,
      false,
    );
  }, []);

  // Fill height from progress (liquid rises from bottom)
  const fillY = useDerivedValue(() => {
    // At progress=0: fill starts at bottom (empty vessel)
    // At progress=1: fill reaches top (full)
    const fillFraction = progress.value;
    // fillY is the Y coordinate of the liquid surface
    // When empty: fillY = cy + r (bottom of circle)
    // When full:  fillY = cy - r (top of circle)
    return cy + r - fillFraction * (2 * r);
  });

  // Wave amplitude — smaller when almost full or almost empty
  const waveAmp = useDerivedValue(() => {
    const p = progress.value;
    // Reduce wave near 0 and 1 for clean entry/exit
    const edgeDamp = Math.min(p * 8, (1 - p) * 8, 1);
    return 6 * edgeDamp;
  });

  // Build the liquid path using Skia's path builder
  const liquidPath = useDerivedValue(() => {
    const path = Skia.Path.Make();
    const steps = 60; // resolution of the wave
    const stepW = (r * 2) / steps;
    const startX = cx - r;

    // Start at bottom-left of circle
    path.moveTo(startX, cy + r);

    // Draw wave across the top surface
    for (let i = 0; i <= steps; i++) {
      const x = startX + i * stepW;
      const waveY = fillY.value
        + waveAmp.value * Math.sin(wavePhase.value + (i / steps) * Math.PI * 4);
      if (i === 0) path.moveTo(x, waveY);
      else path.lineTo(x, waveY);
    }

    // Close the bottom
    path.lineTo(cx + r, cy + r);
    path.lineTo(startX, cy + r);
    path.close();
    return path;
  });

  // Color transitions: 0–25% warm teal → 25–75% indigo → 75–100% gold
  const liquidColor = useDerivedValue(() => {
    const p = progress.value;
    if (p < 0.25) {
      return interpolateColor(p / 0.25, [0, 1], ['#2DD4BF', '#6366F1']);
    } else if (p < 0.75) {
      return interpolateColor((p - 0.25) / 0.5, [0, 1], ['#6366F1', '#8B5CF6']);
    } else {
      // Blend toward the theme's primary gold on final quarter
      return interpolateColor((p - 0.75) / 0.25, [0, 1], ['#8B5CF6', theme.primary]);
    }
  });

  return (
    <View style={{ width: size, height: size }}>
      <Canvas style={{ width: size, height: size }}>
        {/* Circle outline / track */}
        <Circle
          cx={cx} cy={cy} r={r}
          color={theme.cardBorder}
          style="stroke"
          strokeWidth={2}
        />

        {/* Liquid fill clipped to circle */}
        <Group
          clip={Skia.Path.Make().addCircle(cx, cy, r)}
        >
          <Path path={liquidPath} color={liquidColor} opacity={0.85} />
          {/* Second wave layer, offset phase for depth illusion */}
          <Path path={liquidPath} color={liquidColor} opacity={0.35}
            transform={[{ translateY: 4 }]}
          />
        </Group>

        {/* Circle outline on top (for clean edge) */}
        <Circle
          cx={cx} cy={cy} r={r}
          color={theme.cardBorder}
          style="stroke"
          strokeWidth={1.5}
        />
      </Canvas>
    </View>
  );
}
```

#### Step 2: The Mandala Progression System — `components/meditation/MandalaView.tsx`

This is a Skia drawing that incrementally reveals geometry as `progress` goes from 0 to 1. Each meditation type has its own geometry:

```typescript
// components/meditation/MandalaView.tsx
import React from 'react';
import { View } from 'react-native';
import { Canvas, Path, Circle, Group, Paint,
         Skia, useDerivedValue } from '@shopify/react-native-skia';
import type { SharedValue } from 'react-native-reanimated';

type MeditationType = 'mindfulness' | 'vipassana' | 'yoga-nidra' | 'trataka' | null;

interface MandalaViewProps {
  progress: SharedValue<number>; // 0→1
  meditationType: MeditationType | string | null;
  size?: number;
  theme: {
    primary: string;
    primaryLight: string;
    textMuted: string;
    orb: string;
  };
}

// ---- Utility: build lotus petal path ----
function buildLotusPetal(
  cx: number, cy: number, r: number,
  angleRad: number, petalLength: number, petalWidth: number
): ReturnType<typeof Skia.Path.Make> {
  const path = Skia.Path.Make();
  // Petal is a rotated ellipse extending outward from center
  const tipX = cx + Math.cos(angleRad) * (r + petalLength);
  const tipY = cy + Math.sin(angleRad) * (r + petalLength);
  const leftX = cx + Math.cos(angleRad + Math.PI / 2) * petalWidth;
  const leftY = cy + Math.sin(angleRad + Math.PI / 2) * petalWidth;
  const rightX = cx + Math.cos(angleRad - Math.PI / 2) * petalWidth;
  const rightY = cy + Math.sin(angleRad - Math.PI / 2) * petalWidth;

  path.moveTo(cx + Math.cos(angleRad) * r, cy + Math.sin(angleRad) * r);
  path.cubicTo(leftX, leftY, tipX - 5, tipY - 5, tipX, tipY);
  path.cubicTo(tipX + 5, tipY + 5, rightX, rightY, cx + Math.cos(angleRad) * r, cy + Math.sin(angleRad) * r);
  path.close();
  return path;
}

export default function MandalaView({
  progress, meditationType, size = 280, theme,
}: MandalaViewProps) {
  const cx = size / 2;
  const cy = size / 2;

  // Determine how many "rings" to show based on progress
  // Each ring has a reveal threshold (0–1)
  // Rings are drawn from innermost outward

  // ---- MINDFULNESS: Lotus (8 petals in 2 layers) ----
  const mindfulnessPaths = useDerivedValue(() => {
    const p = progress.value;
    const paths: { path: ReturnType<typeof Skia.Path.Make>; opacity: number }[] = [];

    // Center seed (always visible from start)
    const seed = Skia.Path.Make();
    seed.addCircle(cx, cy, 8);
    paths.push({ path: seed, opacity: Math.min(p * 10, 1) });

    // Inner ring of 4 petals — appears 0–0.3
    if (p > 0.05) {
      for (let i = 0; i < 4; i++) {
        const angle = (i / 4) * Math.PI * 2;
        const reveal = Math.min((p - 0.05) / 0.25, 1);
        const petal = buildLotusPetal(cx, cy, 20, angle, 30 * reveal, 12);
        paths.push({ path: petal, opacity: reveal * 0.85 });
      }
    }

    // Outer ring of 8 petals — appears 0.3–0.7
    if (p > 0.3) {
      for (let i = 0; i < 8; i++) {
        const angle = (i / 8) * Math.PI * 2;
        const reveal = Math.min((p - 0.3) / 0.4, 1);
        const petal = buildLotusPetal(cx, cy, 45, angle, 45 * reveal, 16);
        paths.push({ path: petal, opacity: reveal * 0.65 });
      }
    }

    // Outer circle border — appears 0.7–1.0
    if (p > 0.7) {
      const ring = Skia.Path.Make();
      const reveal = Math.min((p - 0.7) / 0.3, 1);
      ring.addCircle(cx, cy, 90 * reveal + 5);
      paths.push({ path: ring, opacity: reveal * 0.4 });
    }

    return paths;
  });

  // ---- YOGA NIDRA: Concentric ripples ----
  const yogaNidraPaths = useDerivedValue(() => {
    const p = progress.value;
    const paths: { r: number; opacity: number }[] = [];
    const ringCount = 8;
    for (let i = 0; i < ringCount; i++) {
      const threshold = i / ringCount;
      if (p > threshold) {
        const reveal = Math.min((p - threshold) / (1 / ringCount), 1);
        paths.push({
          r: 20 + i * 14,
          opacity: reveal * (0.6 - i * 0.06),
        });
      }
    }
    return paths;
  });

  // ---- VIPASSANA: Geometric — hexagons and triangles ----
  const vipassanaPaths = useDerivedValue(() => {
    const p = progress.value;
    const paths: { path: ReturnType<typeof Skia.Path.Make>; opacity: number }[] = [];

    // Center triangle
    if (p > 0) {
      const tri = Skia.Path.Make();
      const triSize = 25 * Math.min(p * 5, 1);
      tri.moveTo(cx, cy - triSize);
      tri.lineTo(cx + triSize * 0.866, cy + triSize * 0.5);
      tri.lineTo(cx - triSize * 0.866, cy + triSize * 0.5);
      tri.close();
      paths.push({ path: tri, opacity: Math.min(p * 5, 1) * 0.8 });
    }

    // Hexagon at 0.2
    if (p > 0.2) {
      const hex = Skia.Path.Make();
      const hexR = 50 * Math.min((p - 0.2) / 0.3, 1);
      for (let i = 0; i < 6; i++) {
        const angle = (i / 6) * Math.PI * 2 - Math.PI / 6;
        const x = cx + hexR * Math.cos(angle);
        const y = cy + hexR * Math.sin(angle);
        if (i === 0) hex.moveTo(x, y);
        else hex.lineTo(x, y);
      }
      hex.close();
      paths.push({ path: hex, opacity: Math.min((p - 0.2) / 0.3, 1) * 0.5 });
    }

    // Outer star of David at 0.5
    if (p > 0.5) {
      const star = Skia.Path.Make();
      const starR = 90 * Math.min((p - 0.5) / 0.5, 1);
      // Two overlapping triangles
      for (let tri = 0; tri < 2; tri++) {
        const offset = tri * (Math.PI / 3);
        for (let i = 0; i < 3; i++) {
          const angle = (i / 3) * Math.PI * 2 + offset;
          const x = cx + starR * Math.cos(angle);
          const y = cy + starR * Math.sin(angle);
          if (i === 0) star.moveTo(x, y);
          else star.lineTo(x, y);
        }
        star.close();
      }
      paths.push({ path: star, opacity: Math.min((p - 0.5) / 0.5, 1) * 0.35 });
    }

    return paths;
  });

  const renderMandala = () => {
    const type = meditationType ?? 'mindfulness';

    if (type === 'yoga-nidra') {
      return (
        <>
          {yogaNidraPaths.value.map((ring, i) => (
            <Circle
              key={i}
              cx={cx} cy={cy} r={ring.r}
              color={theme.primary}
              opacity={ring.opacity}
              style="stroke"
              strokeWidth={1.5}
            />
          ))}
        </>
      );
    }

    if (type === 'vipassana') {
      return (
        <>
          {vipassanaPaths.value.map((item, i) => (
            <Path
              key={i}
              path={item.path}
              color={theme.textMuted}
              opacity={item.opacity}
              style="stroke"
              strokeWidth={1}
            />
          ))}
        </>
      );
    }

    // Default: mindfulness lotus
    return (
      <>
        {mindfulnessPaths.value.map((item, i) => (
          <Path
            key={i}
            path={item.path}
            color={theme.primary}
            opacity={item.opacity}
            style={i === 0 ? 'fill' : 'stroke'}
            strokeWidth={1.5}
          />
        ))}
      </>
    );
  };

  return (
```

```typescript
// components/focus/AmbientWave.tsx
import React from 'react';
import { View } from 'react-native';
import { Canvas, Path, Skia, useDerivedValue } from '@shopify/react-native-skia';
import { useSharedValue, withRepeat, withTiming,
         useEffect as useReanimatedEffect } from 'react-native-reanimated';

interface AmbientWaveProps {
  width: number;
  color: string;
}

export default function AmbientWave({ width, color }: AmbientWaveProps) {
  const CANVAS_H = 80;
  const wavePhase = useSharedValue(0);
  const amplitude = useSharedValue(8);

  useReanimatedEffect(() => {
    // Phase scrolls continuously
    wavePhase.value = withRepeat(
      withTiming(Math.PI * 2, { duration: 4000 }),
      -1, false,
    );

    // Amplitude breathes on a 4-second cycle
    amplitude.value = withRepeat(
      withTiming(14, { duration: 4000 }),
      -1, true, // reverse = true for back-and-forth
    );
  }, []);

  const wavePath = useDerivedValue(() => {
    const path = Skia.Path.Make();
    const steps = 80;
    const stepW = width / steps;
    const baseY = CANVAS_H / 2;

    path.moveTo(0, baseY);
    for (let i = 1; i <= steps; i++) {
      const x = i * stepW;
      const y = baseY + amplitude.value * Math.sin(wavePhase.value + (i / steps) * Math.PI * 4);
      path.lineTo(x, y);
    }
    return path;
  });

  return (
    <View pointerEvents="none" style={{ width, height: CANVAS_H }}>
      <Canvas style={{ width, height: CANVAS_H }}>
        <Path
          path={wavePath}
          color={color}
          opacity={0.10}
          style="stroke"
          strokeWidth={1.5}
        />
      </Canvas>
    </View>
  );
}
```

```typescript
const SFX_TRACKS = {
  // EXISTING:
  'ding-soft':       require('../assets/audio/sfx/ding-soft.mp3'),
  'singing-bowl':    require('../assets/audio/sfx/singing-bowl.mp3'),
  'breath-complete': require('../assets/audio/sfx/breath-complete.mp3'),

  // NEW (S9):
  'region-select':   require('../assets/audio/sfx/sfx-region-select.mp3'),
  'session-begin':   require('../assets/audio/sfx/sfx-session-begin.mp3'),
  'phase-transition':require('../assets/audio/sfx/sfx-phase-transition.mp3'),
  'session-end':     require('../assets/audio/sfx/sfx-session-end.mp3'),
  'streak-milestone':require('../assets/audio/sfx/sfx-streak-milestone.mp3'),
  'badge-earned':    require('../assets/audio/sfx/sfx-badge-earned.mp3'),
};

// Expand useSFX() return value:
export function useSFX() {
  const { sfxVolume } = useSettings();

  const play = async (trackKey: keyof typeof SFX_TRACKS, volumeMultiplier = 1.0) => {
    if (sfxVolume === 0) return;
    try {
      const { sound } = await Audio.Sound.createAsync(SFX_TRACKS[trackKey]);
      await sound.setVolumeAsync(sfxVolume * volumeMultiplier);
      await sound.playAsync();
      // Auto-unload after playback to prevent memory leak
      sound.setOnPlaybackStatusUpdate((status) => {
        if (status.isLoaded && status.didJustFinish) {
          sound.unloadAsync();
        }
      });
    } catch (e) {
      // Fail silently — SFX is enhancement, not critical
    }
  };

  return {
    // Existing:
    playDing:         () => play('ding-soft'),
    playSingingBowl:  () => play('singing-bowl'),
    playBreathDone:   () => play('breath-complete'),

    // New S9:
    playRegionSelect:   () => play('region-select'),
    playSessionBegin:   () => play('session-begin'),
    playPhaseTransition:() => play('phase-transition', 0.4), // 40% volume — subtle
    playSessionEnd:     () => play('session-end'),
    playStreakMilestone:() => play('streak-milestone'),
    playBadgeEarned:    () => play('badge-earned'),
  };
}
```

```typescript
// NEW S9: Phase-specific haptic for breathing
// Call with the phase name that is STARTING
export async function hapticForBreathingPhase(phase: string) {
  if (!isHapticsEnabled()) return;
  switch (phase) {
    case 'inhale':
      await hapticRisingSequence(); // "Begin"
      break;
    case 'holdIn':
```

```typescript
// store/historyStore.ts
import { create } from 'zustand';
import { persist, createJSONStorage } from 'zustand/middleware';
import AsyncStorage from '@react-native-async-storage/async-storage';

export interface SessionRecord {
  id: string;                    // unique: timestamp string
  completedAt: string;           // ISO date string
  type: 'breathing' | 'meditation' | 'focus' | 'pomodoro';
  durationMinutes: number;
  theme: string;                 // e.g. 'rajasthan'
  practiceId: string | null;     // e.g. 'box-breathing', 'mindfulness'
  intention: string | null;
  moodBefore: number | null;     // 1-5
  moodAfter: number | null;      // 1-5
  accomplishmentNote: string | null;
}

interface HistoryState {
  sessions: SessionRecord[];
  earnedBadgeIds: string[];
  
  // Actions
  addSession: (record: Omit<SessionRecord, 'id' | 'completedAt'>) => void;
  awardBadge: (badgeId: string) => void;
  
  // Derived (computed on demand — don't store, compute)
  getStreak: () => number;
  getTotalSessions: () => number;
  getTotalMinutes: () => number;
  getMalaProgress: () => number;  // 0–107
  getSessionsForTheme: (theme: string) => number;
}

export const useHistoryStore = create<HistoryState>()(
  persist(
    (set, get) => ({
      sessions: [],
      earnedBadgeIds: [],

      addSession: (record) => {
        const newRecord: SessionRecord = {
          ...record,
          id: Date.now().toString(),
          completedAt: new Date().toISOString(),
        };
        set((s) => ({ sessions: [newRecord, ...s.sessions] }));
      },

      awardBadge: (badgeId) => {
        const already = get().earnedBadgeIds.includes(badgeId);
        if (!already) {
          set((s) => ({ earnedBadgeIds: [...s.earnedBadgeIds, badgeId] }));
        }
      },

      getStreak: () => {
        const { sessions } = get();
        if (sessions.length === 0) return 0;

        // Get unique calendar days with sessions, sorted descending
        const days = new Set(
          sessions.map((s) => s.completedAt.substring(0, 10)) // YYYY-MM-DD
        );
        const sortedDays = Array.from(days).sort().reverse();

        const today = new Date().toISOString().substring(0, 10);
        const yesterday = new Date(Date.now() - 86400000).toISOString().substring(0, 10);

        // Streak only counts if today OR yesterday has a session
        if (sortedDays[0] !== today && sortedDays[0] !== yesterday) return 0;

        let streak = 0;
        let checkDate = new Date(sortedDays[0]);

        for (const day of sortedDays) {
          const dayDate = new Date(day);
          const diffMs = checkDate.getTime() - dayDate.getTime();
          const diffDays = Math.round(diffMs / 86400000);
          if (diffDays > 1) break;
          streak++;
          checkDate = dayDate;
        }
        return streak;
      },

      getTotalSessions: () => get().sessions.length,
      getTotalMinutes: () => get().sessions.reduce((sum, s) => sum + s.durationMinutes, 0),
      getMalaProgress: () => get().sessions.length % 108,
      getSessionsForTheme: (theme) =>
        get().sessions.filter((s) => s.theme === theme).length,
    }),
    {
      name: 'flowsome:history',
      storage: createJSONStorage(() => AsyncStorage),
    }
  )
);
```

```typescript
            color={bead.lit ? theme.primary : theme.cardBorder}
```

```typescript
// utils/achievementUtils.ts
import { ACHIEVEMENTS } from '../constants/achievements';
import type { AchievementStats } from '../constants/achievements';
import { useHistoryStore } from '../store/historyStore';

// Compute current stats from history store (call after every session)
export function computeAchievementStats(): AchievementStats {
  const store = useHistoryStore.getState();
  const { sessions } = store;

  const sessionsByTheme: Record<string, number> = {};
  const sessionsByType: Record<string, number> = {};
  let morningCount = 0;
  let nightCount = 0;

  for (const session of sessions) {
    // By theme
    sessionsByTheme[session.theme] = (sessionsByTheme[session.theme] ?? 0) + 1;
    // By type
    sessionsByType[session.type] = (sessionsByType[session.type] ?? 0) + 1;
    // Time of day
    const hour = new Date(session.completedAt).getHours();
    if (hour < 7) morningCount++;
    if (hour >= 22) nightCount++;
  }

  const hasCompletedDeepWork90 = sessions.some(
    (s) => s.type === 'focus' && s.durationMinutes >= 90
  );
  const hasCompletedMeditation30 = sessions.some(
    (s) => s.type === 'meditation' && s.durationMinutes >= 30
  );

  return {
    totalSessions: sessions.length,
    streak: store.getStreak(),
    sessionsByTheme,
    sessionsByType,
    totalMinutes: store.getTotalMinutes(),
    hasCompletedDeepWork90,
    hasCompletedMeditation30,
    morningSessionCount: morningCount,
    nightSessionCount: nightCount,
    breathingSessions: sessionsByType['breathing'] ?? 0,
  };
}

// Check all unearned badges and award newly qualifying ones.
// showToast callback is passed from the screen component.
export function checkAndAwardBadges(
  awardBadge: (id: string) => void,
  showToast: (badge: typeof ACHIEVEMENTS[0]) => void,
) {
  const { earnedBadgeIds } = useHistoryStore.getState();
  const stats = computeAchievementStats();

  for (const achievement of ACHIEVEMENTS) {
    if (earnedBadgeIds.includes(achievement.id)) continue;
    if (achievement.checkCondition(stats)) {
      awardBadge(achievement.id);
      showToast(achievement);
    }
  }
}
```

```typescript
// components/stats/HeatmapCalendar.tsx
import React, { useMemo } from 'react';
import { View, useWindowDimensions } from 'react-native';
import { Canvas, RoundedRect } from '@shopify/react-native-skia';
import type { SessionRecord } from '../../store/historyStore';

const THEME_COLORS: Record<string, string> = {
  rajasthan: '#F59E0B',
  himalaya: '#60A5FA',
  kerala:   '#34D399',
  assam:    '#A3E635',
  andaman:  '#38BDF8',
};

interface HeatmapCalendarProps {
  sessions: SessionRecord[];
  theme: { cardBorder: string; background: string; text: string };
}

export default function HeatmapCalendar({ sessions, theme }: HeatmapCalendarProps) {
  const { width } = useWindowDimensions();
  const COLS = 53; // weeks
  const ROWS = 7;  // days of week
  const CELL = Math.floor((width - 40) / COLS);
  const GAP = 1;
  const CANVAS_W = COLS * (CELL + GAP);
  const CANVAS_H = ROWS * (CELL + GAP);

  // Build a map of date → { totalMinutes, dominantTheme }
  const heatmap = useMemo(() => {
    const map: Record<string, { minutes: number; theme: string }> = {};
    for (const session of sessions) {
      const day = session.completedAt.substring(0, 10);
      if (!map[day]) map[day] = { minutes: 0, theme: session.theme };
      map[day].minutes += session.durationMinutes;
      // Keep the theme with most minutes (first occurrence wins for simplicity)
    }
    return map;
  }, [sessions]);

  // Build 53×7 grid of cells. Start from 52 weeks ago (364 days + today).
  const cells = useMemo(() => {
    const today = new Date();
    const cells: { date: string; col: number; row: number }[] = [];
    // Start date: 52 full weeks before today's day-of-week
    const startDate = new Date(today);
    startDate.setDate(today.getDate() - 364);
    // Align to Sunday
    startDate.setDate(startDate.getDate() - startDate.getDay());

    for (let col = 0; col < COLS; col++) {
      for (let row = 0; row < ROWS; row++) {
        const d = new Date(startDate);
        d.setDate(startDate.getDate() + col * 7 + row);
        cells.push({
          date: d.toISOString().substring(0, 10),
          col,
          row,
        });
      }
    }
    return cells;
  }, []);

  return (
    <View style={{ width: CANVAS_W, height: CANVAS_H }}>
      <Canvas style={{ width: CANVAS_W, height: CANVAS_H }}>
        {cells.map((cell, i) => {
          const data = heatmap[cell.date];
          const minutes = data?.minutes ?? 0;
          const themeColor = data ? (THEME_COLORS[data.theme] ?? '#6366F1') : 'transparent';

          // Intensity 0–1 from minutes
          const intensity = Math.min(minutes / 60, 1);
          // Mix theme color at intensity over base border color
          const opacity = minutes === 0 ? 0 : 0.15 + intensity * 0.85;

          return (
            <RoundedRect
              key={i}
              x={cell.col * (CELL + GAP)}
              y={cell.row * (CELL + GAP)}
              width={CELL}
              height={CELL}
              r={1.5}
              color={minutes === 0 ? theme.cardBorder : themeColor}
              opacity={minutes === 0 ? 0.15 : opacity}
            />
          );
        })}
      </Canvas>
    </View>
  );
}
```

```typescript
export function getTimeGreeting(streakDays = 0): GreetingResult {
  const hour = new Date().getHours();
  const normalizedHour = hour < 5 ? hour + 24 : hour;

  const match = GREETINGS.find(
    ([from, to]) => normalizedHour >= from && normalizedHour < to,
  ) ?? GREETINGS[GREETINGS.length - 1];

  // ↑ The destructuring pattern here should use the object:
  const entry = GREETINGS.find(
    (g) => normalizedHour >= g.hours[0] && normalizedHour < g.hours[1],
  ) ?? GREETINGS[GREETINGS.length - 1];

  const streakSuffix = streakDays > 0 ? ` · Day ${streakDays} 🪔` : '';
  return {
    line: entry.line + streakSuffix,
    lineHindi: entry.lineHindi,
  };
```

```typescript
  { start: [1, 25], end: [2, 5],  name: 'Holi',       line: 'Holi · Colour and renewal.' },
  { start: [8, 19], end: [8, 28], name: 'Navratri',   line: 'Navratri · Nine nights of the divine.' },
];

const SEASONS = [
  { months: [2, 3],    name: 'Vasanta', nameHindi: 'वसंत',    line: 'The petals are opening.' },
  { months: [4, 5],    name: 'Grishma', nameHindi: 'ग्रीष्म',  line: 'The sun teaches endurance.' },
  { months: [6, 7],    name: 'Varsha',  nameHindi: 'वर्षा',   line: 'The earth is being washed clean.' },
  { months: [8, 9],    name: 'Sharad',  nameHindi: 'शरद्',    line: 'The harvest of stillness.' },
  { months: [10, 11],  name: 'Hemanta', nameHindi: 'हेमंत',   line: 'The fog that precedes clarity.' },
  { months: [0, 1],    name: 'Shishira',nameHindi: 'शिशिर',   line: 'The cold that deepens the root.' },
];

export function getCurrentSeasonInfo(): SeasonInfo {
  const now = new Date();
  const month = now.getMonth();
  const day = now.getDate();

  // Check festivals first
  for (const festival of FESTIVALS) {
    const [sm, sd] = festival.start;
    const [em, ed] = festival.end;
    const afterStart = month > sm || (month === sm && day >= sd);
    const beforeEnd  = month < em || (month === em && day <= ed);
    if (afterStart && beforeEnd) {
      return { name: festival.name, nameHindi: '', line: festival.line };
    }
  }

  // Fall back to season
  const season = SEASONS.find((s) => s.months.includes(month)) ?? SEASONS[4];
  return { name: season.name, nameHindi: season.nameHindi, line: season.line };
}
```

### 13B — FINAL BUG SWEEP (Common Expo 56 Issues)

**Issue: RadialGradient in Skia not accepting SharedValue for colors**

Skia's `RadialGradient` in version 2.6.2 may not support `useDerivedValue` for the `colors` array directly. If you see errors, use a workaround:

```typescript
// Workaround: Use two overlapping circles with different opacities
// Instead of gradient, manually simulate with:
<Circle cx={cx} cy={cy} r={orbRadius} color={theme.orb} />
<Circle cx={cx} cy={cy} r={orbRadius} color={centerColor} opacity={0.7} />
// centerColor is from useDerivedValue returning static strings
```

**Issue: `useEffect as useReanimatedEffect` import**

In Reanimated 4.3.1, the import is:
```typescript
import { useAnimatedReaction } from 'react-native-reanimated';
// useEffect from 'react-native-reanimated' is:
import Animated, { useSharedValue, runOnJS } from 'react-native-reanimated';
// For starting animations on mount, use the standard pattern:
// In a regular useEffect, just set .value with withRepeat/withTiming
```

The `useEffect as useReanimatedEffect` pattern shown in S6/S7 requires Reanimated's own `useEffect`. If it's not exported in your version:
```typescript
// Replace:
// useReanimatedEffect(() => { ... }, []);
// With standard React useEffect:
useEffect(() => {
  animSharedValue.value = withRepeat(withTiming(...), -1, false);
  return () => {
    cancelAnimation(animSharedValue);
  };
}, []);
```

**Issue: `expo-brightness` permissions on Android**

`expo-brightness` requires the `WRITE_SETTINGS` permission on Android. In SDK 56 managed workflow, you may need to add to `app.json`:
```json
{
  "expo": {
    "plugins": [
      ["expo-brightness", {
        "requestedPermissions": ["WRITE_SETTINGS"]
      }]
    ]
  }
}
```
Test this: if `setBrightnessAsync` fails silently, catch the error and skip brightness adjustment gracefully — it's enhancement, not critical.

**Issue: `eas build` fails with Skia native code**

If EAS build fails on Skia:
1. Verify `@shopify/react-native-skia` is exactly `2.6.2` in package.json (not `^2.6.2`)
2. Check eas.json has `"buildType": "apk"` for preview builds
3. If using `npx expo prebuild` → verify `android/app/build.gradle` has Skia's CMake config

**Issue: NowPlayingBar flickers when navigating**

If NowPlayingBar disappears briefly during navigation:
```typescript
// In _layout.tsx, ensure it renders OUTSIDE the Stack, not inside:
return (
  <GestureHandlerRootView style={{ flex: 1 }}>
    <SafeAreaProvider>
      <Stack>...</Stack>
      <NowPlayingBar /> {/* This is outside Stack, stays mounted during nav */}
    </SafeAreaProvider>
  </GestureHandlerRootView>
);
```
